# Mirau's Dotfiles
Mirau's current used .config files for a consistent Linux desktop experience.


## Dependencies
* i3
* polybar
* rofi
* kitty
* Neofetch


## Installation
Your standard stuff. Each config has a pretty standard install location.
* i3 `~/.config/i3/`
* Polybar `~/.config/polybar/`
* Rofi `~/.config/rofi/`
* Kitty `~/.config/kitty`
* Neofetch `~/.config/neofetch`
* Bash Aliases `~/.bashrc`
